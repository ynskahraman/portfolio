# Yunus Emre Kahraman — 3D Portfolio

[Bruno Simon](https://bruno-simon.com/) tarzında, arabayla gezilen interaktif portföy.

## Başlat

```bash
npm install
npm run dev
```

## Harita

| Bölge | Konum | İçerik |
|-------|-------|--------|
| Hoş geldin | Merkez | Başlangıç |
| Hakkımda | Sol üst | Bio, eğitim |
| Yetkinlikler | Sağ üst | Tech stack |
| Deneyim | Sol alt | İş & liderlik |
| İletişim | Sağ alt | E-posta, sosyal |
| Projeler | Kuzey yol | Within Air, Flowelie, … |

## Kontroller

- **WASD / Ok** — Hareket
- **Shift** — Hızlan
- **Ctrl** — Fren
- **R** — Başlangıca dön

Canlı projelere (Within Air) yaklaşınca site otomatik açılır.

## Özelleştirme

`src/data/profile.ts` — CV verileri, projeler, zonlar.
